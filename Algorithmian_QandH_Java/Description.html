우선순위 큐에서는 항목들이 추가되는 순서와 처리되는 순서가 다를 수 있다 먼저 들어온 것이 먼저 처리되지 않고 정해진 우선순위에 따라 처리되는 작업 스케줄링이 우선순위 큐이다 오름차순 우선순위 큐 : 작은 키 값을 갖는 항목일수록 높은 우선순위를 갖는 우선순위 큐(즉 항상 제일 작은 항목을 삭제) 내림차순 우선순위 큐 : 큰 키 값을 갖는 항목일수록 높은 우선순위를 갖는 우선순위 큐(즉 항상 제일 큰 항목을 삭제) 힙이란 무엇인가? 힙은 몇가지 특수한 속성을 가진 트리이다. 힙의 기본 요구사항은 노드의 값이 그 자식노드의 값보다 >= or <=한다는 것이다 또한 완전 이진트리여야한다 1. 최소 힙 : 노드의 값이 자식 노드의 값보다 작거나 같아야만 한다 2. 최대 힙 : 노드의 값이 자식 노드의 값보다 크거나 같아야만 한다
